# Covid19Website

A static informational website about **COVID-19**, built from an Adobe XD design (`Designs/Web design.xd`) and hand-coded in HTML, CSS and vanilla JavaScript.

Illustrations come from the [unDraw](https://undraw.co/) medical / doctor / research sets.

## Contents

- `index.html` — the landing page markup.
- `style.css` — hand-written styles.
- `script.js`, `script2.js` — vanilla JS interactions.
- `Designs/` — Adobe XD source file and exported PNGs of the three main breakpoints (`Web 1920 – 1/2/3.png`).
- `Images/` — unDraw illustrations at 1× and 2× resolutions (doctor, medical research, medicine).

## Running

Open `index.html` directly in a browser, or serve the folder with any static file server:

```bash
python -m http.server
```

Then visit `http://localhost:8000`.
